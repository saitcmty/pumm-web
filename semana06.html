Introducción a JavaScript - Parte 2

¿De qué trata la sesión?

JavaScript (abreviado como JS) es un lenguaje de programación de alto nivel. Junto con HTML y CSS, JavaScript es una de las tecnologías centrales de la World Wide Web (WWW). JavaScript permite crear páginas web interactivas y es una parte escencial de las aplicaciones web. La mayor parte de los sitios web lo utilizan para el comportamiento en el lado del cliente y casi todos los navegadores web (browsers) están diseñados para poder ejecutar código escrito en JavaScript.
