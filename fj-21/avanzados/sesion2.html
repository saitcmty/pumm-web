Web Development 101

¿De qué trata la sesión?

En esta clase, se introducen los conceptos básicos del desarollo web. Se habla sobre lo que son HTML, CSS y JavaScript y cómo nos ayudan a crear páginas web atractivas visualmente e interactivas. Asimismo, se introduce el concepto de "framework" y la importancia de HTML, CSS y JavaScript para sacarles el mayor provecho. Por último, mostramos el ejemplo de Bootstrap, siendo éste el framework de CSS más usado para construir páginas web rápidamente.
